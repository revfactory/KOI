- 수업 시간 충돌: `source.txt`는 19:00-21:00 2시간 일정과 90분 강의 + 60분 멘토링 모델을 함께 제시한다. 문서에는 2시간 압축안과 별도 멘토링 확장 가능성을 명시했다.
- 팀 규모: 원문에 팀 규모가 없어 문서에는 역할 기준만 제시하고 인원 수는 고정하지 않았다.

- AI 도구 허용 범위: 특정 도구를 필수로 지정하지 않고 AI 사용 로그와 인간 검증 기준을 필수로 두었다.
- 외부 PR 미병합: 병합 여부보다 Issue, PR, Review, 재현, 수정 기록을 인정하는 운영 기준으로 정리했다.
